Why should one entrust server administration to us?

We will take care of all your problems. All our clients get free basic administration package.

Do your things and don’t worry about technical aspects.

The service of free basic administration

includes the following works carried out by ProfitServer technical support specialists:

  • Initial installation of operating system (OS) at client’s choice (within the framework of the list of OS available for installation for chosen tariff);
  • Reinstallation of OS at client’s choice (without data preservation);
  • Virtual server reboot at client’s choice;
  • Adding of additional purchased IP-addresses;
  • Data backup adjustment (only in the case if a client had purchased the “space for backing up” service on ProfitServer’s backup server);
  • Transfer of sites from VDS to a dedicated server purchased by client on ProfitServer resources.
Any administration package
DOES NOT include the following works:

Training clients to Linux, FreeBSD, Windows operating systems’ administration basics.

Adjustment and maintenance of software operability of game servers, proxi and other specific software installed by a client or specialists of ProfitServer within the frameworks of paid request.

Works on the search and elimination of errors in scripts of client’s software.

Works on the search and elimination of errors in SQL queries and also on their optimization.

Advanced administration package service

includes the following works carried out by ProfitServer technical support specialists:

  • All types of free basic administration works (the number of requests is not added to the requests within the framework of advanced package);
  • Installation of virtual server control panel ISPManager 5;
  • Installation of main services (PHP, FTP, Apache, MySQL, etc.) at client’s request;
  • Making of necessary changes in services’ configuration files, change of operating system’s sets;
  • Setting of data backup schedule in accordance with client’s requirements (only in the case if a client had purchased the “space for backing up” service on ProfitServer’s backup server);
  • Optimizing of virtual/dedicated server’s work;
  • Installation of additional modules and extensions for services (PHP, Apache, etc.);
  • Checking of server for virus software at client’s request;
  • Adding of server to ProfitServer monitoring system;
  • Analysis of system’s log-files for search and elimination of problems and their causes;
  • Applying of basic software upgrades recommended by manufacturer for the reasons of safety (hotfixes) if necessary;
  • Solving of problems in case they’re detected before contacting technical support.
  • Reset of administration password of operating system (for VDS service);
